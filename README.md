## Backend Setup

### 1. Create `.env` file inside `backend/` with:
MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

PORT=5000

### 2. Install dependencies
npm install

### 3. Start backend server
npm start

## Frontend Setup

### 1. Install dependencies
cd frontend
npm install

### 2. Start development server
npm run dev
